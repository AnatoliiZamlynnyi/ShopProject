							СИСТЕМА "МАГАЗИН"
	Призначена для власників невеликих магазинів яка полегшує введення балансу, проведення ревізій та контроль за наявністю товару на момент ревізій.
			Рівні доступа: (існує два рівні доступа)
	Адміністратор: власник магазину або уповноважено-відповідальна особа за весь товар та ведення торгівлі, має права приймати товар, списувати та продавати.
	Користувач: продавець магазину, має права на прийом товару та продаж.
			Функціонал системи:
	Інтуітивно зрозумілий інтерфейс, кнопки "Оновити" перезавантажують той розділ під яким знаходяться плюс конопка "Оновити" під розділом товару скидає налаштування проміжку дати. Кнопки "Додати" додають елемент в певну категорію.
	Редагування можливе так би мовити "на льоту": клік лівою кнопкою миші по тій позицію яку хочемо змінити в потрібному елементу, F2 та редагуємо,  по завершеню редагування, для прийняття змін, тиснемо клавішу Enter.
	Якщо потрібно видалити елемент то клікаємо по потрібному елементу та тиснемо комбінацію клавіш правийCtrl+Del. 
*Застереження: видаленний елемент повернути неможливо. Видаляти може тільки Адміністратор.
	Подвійний клік по певній категорії товару або постачальнику призводить відбір товару, а якщо встановлений діапазон дат то за певний період.
